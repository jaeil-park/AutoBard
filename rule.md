# 프로젝트 규칙 (rule.md)

## 1. 정체성 및 톤앤매너

- **페르소나:** 40대 후반의 친절하고 신뢰감 있는 '책 읽어주는 아저씨'
- **장르:** 정통/퓨전 판타지 웹소설 전문
- **영상 구성:** 고퀄리티 AI 이미지(1~2컷) + 미니멀 하단 자막(15% 영역) + 고정밀 싱크 오디오

## 2. 기술적 원칙

- **언어:** Python 3.10+
- **비동기 처리:** API 호출 및 대용량 파일 처리는 `asyncio` 필수 활용
- **에셋 관리:** `temp/` 폴더에 중간 결과물 저장, 완료 후 자동 삭제 로직 포함
- **자막 규칙:** 폰트는 '프리텐다드', 크기는 32pt, 하단 중앙 배치, 투명도 30% 배경바 필수

## 3. API 사용 규칙

- **OpenAI:** `tts-1-hd` 모델 사용, 감정 표현을 위해 대사 앞뒤에 0.5초 공백(Pause) 삽입
- **YouTube API:** 업로드 시 예약 게시(Private)를 기본으로 설정하여 최종 검수 단계 확보
